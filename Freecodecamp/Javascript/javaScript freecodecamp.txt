-camelCase : 
	En camelCase , los nombres de variables de 
	varias palabras tienen la primera palabra en minúsculas 
	y la primera letra de cada palabra subsiguiente está en mayúscula.

.length : para mostrar la longitud de una cadena

notacion de corchetes: Para hallar el primer caracter 
[0] =  obtener el primer caracter o la primera posicion
